    Parameters:
    Name Type Description
    element Element The element associated with the view
    options Object Options for the view
    Name Type Description
    name string | null = null - The name of the view. You should set this
    persist boolean = false - If a view is persistant, it will not exit if it's not found within the loaded document.
    transition Transition = Transition - The transition to use for this view
    selector string | null = null - The selector to retreive a new node for this view from the loaded document. If left set to `null`, `[data-view="viewname"]` will be used. If you set an attributeOverride for 'data-view', that will be used instead.
    model Model | null = null - The initial model of the view. You should set this.
