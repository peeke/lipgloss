    Transition

    Basic Transition class. All transitions you create should have this class as (grand)parent. When extending Transition, as a rule of thumb it's good to call this the super functions before your own functionality. E.g.: Sometimes you may want to do some preperation work, in which it is just fine to do this before you call super.exit()
